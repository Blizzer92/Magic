# Magic
A game engine for little 2D-Platformer and 2D-RPGs.

The engine is still under heavy development!

## Working

* Rendering
* Font loading
* Texture loading
* Object

## Planning

* Sound management
* Physics
* Collision
* and more awesome things

## Dependencies

Needed to compile the engine:

* SDL2
* SDL2_image
* SDL2_ttf
